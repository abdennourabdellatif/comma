# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-06-16

### Added
- Initial release of Comma
- AI-powered chat interface using Google Gemini API
- Audio/voice processing capabilities with wavesurfer.js
- Firebase authentication and real-time database
- React-based modern UI with Tailwind CSS
- Capacitor integration for Android mobile app
- Multi-language support
- Settings page for user preferences
- Sidebar navigation
- Development server with Vite

### Infrastructure
- GitHub Actions CI/CD pipeline
- Comprehensive .gitignore and .gitattributes
- Contributing guidelines
- Security policy
- MIT License
